# Определение угла сгиба конечности

Веб-приложение для точного определения угла сгиба конечностей в реальном времени с использованием компьютерного зрения.

## Возможности

- Отслеживание углов сгиба для рук и ног в реальном времени
- Визуализация скелета на видео с камеры
- Высокая точность определения углов
- Работает с разной одеждой и даже протезами
- Современный и удобный интерфейс

## Технологии

- **React** - для создания интерфейса
- **TypeScript** - для типобезопасности
- **Vite** - быстрая сборка и разработка
- **MediaPipe Pose** - для определения позы и ключевых точек тела
- **Tailwind CSS** - для стилизации

## Локальная разработка

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для production

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`

## Как использовать

1. Откройте приложение в браузере
2. Разрешите доступ к камере
3. Выберите конечность для отслеживания (левая/правая рука/нога)
4. Встаньте так, чтобы выбранная конечность была полностью видна в кадре
5. Начните сгибать конечность и наблюдайте за изменением угла в реальном времени

## Архитектура

### Компоненты

- **PoseDetector** - основной компонент для работы с камерой и MediaPipe
- **LimbSelector** - выбор отслеживаемой конечности
- **AngleDisplay** - отображение текущего угла сгиба

### Утилиты

- **angleCalculator.ts** - вычисление углов между тремя точками
- **types.ts** - TypeScript типы и интерфейсы
